# Prospecting Playbook: Neuroscience + Psychology + Cold Outreach

This playbook combines your neuroscience framework with cold email best practices and marketing psychology for immediate prospecting use.

---

## THE CORE FRAMEWORK: BRAIN-FIRST PROSPECTING

### 1. The 90/10 Rule (Emotional Decisions, Rational Justification)
**Prospecting Application:** Lead with emotional triggers (pain, fear, desire, status), close with logical proof (ROI, case studies, specs).

```markdown
WRONG: "Our software reduces processing time by 40%"
RIGHT: "Your team is drowning in manual work they hate. We give them their evenings back — and here's the 40% efficiency proof."
```

### 2. System 1 vs System 2 Targeting
| Prospect Stage | Target System | Tactics |
|----------------|---------------|---------|
| Cold outreach | System 1 (fast) | Pattern interrupt, visual proof, social proof, urgency |
| Discovery call | System 1 → 2 transition | SPIN questions, implication stacking |
| Proposal | System 2 (analytical) | ROI calculators, implementation plans, risk reversal |
| Close | System 1 (emotional) | Scarcity, loss aversion, identity reinforcement |

### 3. Contrast Principle in Action
**Price Anchoring Sequence:**
1. Show "cost of inaction" first (lost revenue, wasted hours, competitive disadvantage)
2. Present premium tier (anchors high)
3. Reveal your recommended tier — feels like a bargain
4. Add "starter" tier as decoy (makes recommended look smarter)

## PROSPECTING SEQUENCE: THE 8-SECOND + SPIN HYBRID

### Phase 1: The 8-Second Hook (Email/LinkedIn/Call Opener)

**Structure:** Problem → Visual Proof → Trust Bridge → Low-Friction Ask

```markdown
EMAIL TEMPLATE:
Subject: [Their Company] + [Specific Problem You See]

Hi [Name],

Noticed [specific observation: hiring for X role / posted about Y challenge / using Z tool].

Most [role] at [company size/stage] hit a wall with [specific problem] around now — 
[1-sentence visual proof: "teams waste 12hrs/week on manual reporting" / "competitors cut cycle time 60%"].

We helped [similar company] [specific result with numbers]. 

Worth a 2-minute look at how it applies to you?

[Name]
```

**Key Rules:**
- Subject: 2-4 words, lowercase, internal-looking
- First sentence: Their world, not yours
- Proof: One specific number, one similar company
- CTA: Interest-based ("Worth a look?"), not meeting request

---

### Phase 2: SPIN Discovery (First Call — 15-20 min)

**Don't pitch. Diagnose.**

| SPIN Stage | Questions | Psychological Trigger |
|------------|-----------|----------------------|
| **Situation** | "Walk me through how you handle [process] today" | Commitment (they invest time explaining) |
| **Problem** | "Where does it break?" / "What's the biggest headache?" | Pain activation (System 1 alert) |
| **Implication** | "What happens if this stays unsolved for 6 months?" / "How does this affect [revenue/retention/team morale]?" | Loss aversion, urgency, future pacing |
| **Need-Payoff** | "If you could wave a wand, what would the ideal outcome look like?" / "What would solving this unlock for you?" | Ownership, vision casting, dopamine |

**Implication Stacking Formula:**
```
Problem → Cost (time/money) → Risk (churn/reputation/competitive) → Personal Impact (their bonus/stress/career)
```

---

### Phase 3: The "Visual Proof" Follow-Up (Post-Call)

Send within 2 hours. One page. Three elements:

```markdown
1. THEIR PROBLEM (their words, mirrored back)
2. THE GAP (current state → desired state visual)
3. ONE CASE STUDY: "Company like yours → [specific metric] in [timeframe]"
```

**Format:** Loom video (90 sec) > PDF > Plain text email

---

### Phase 4: Proposal as "Decision Architecture"

Use **Choice Architecture** (Nudge Theory) — don't send one option.

```markdown
OPTION A: Starter (Decoy)
- Core features only
- $X/mo
- "Good if you just need basics"

OPTION B: Growth (TARGET — Anchored against A)
- Everything in A + [key differentiators]
- $Y/mo (only 20% more than A)
- "What most teams your size choose"
- ← Pre-selected / highlighted

OPTION C: Enterprise (Anchor)
- Everything + dedicated support + SLA
- $Z/mo
- "For teams scaling past [threshold]"
```

**Psychology:** Decoy (A) makes Target (B) look rational. Anchor (C) makes Target (B) look safe.

## CHANNEL-SPECIFIC TACTICS

### Cold Email (Primary)
- **Sequence:** 4 emails over 14 days
- **Angles:** Problem → Case Study → Different Problem → Breakup
- **Each email:** Stands alone, adds new value, one CTA

### LinkedIn (Parallel)
- **Connect note:** "Saw your post on [topic] — sharp take on [specific point]"
- **Post-connection:** Value-first content (no pitch), then soft DM
- **Voice notes:** 30 sec, reference their specific content

### Cold Call (High-Intent Only)
- **Opener:** "Hi [Name], [Your Name] from [Company]. We help [specific role] solve [specific problem]. Caught me at a bad time?"
- **Permission-based:** "Can I have 30 seconds to explain why I called?"
- **Transition to SPIN:** "Quick question — how are you handling [problem] today?"

## PSYCHOLOGICAL TRIGGERS CHEAT SHEET

| Trigger | When to Use | Prospecting Application |
|---------|-------------|------------------------|
| **Reciprocity** | First touch | Free audit, benchmark report, relevant template |
| **Commitment** | Discovery | "Would it be useful if I...?" → micro-yes |
| **Social Proof** | All stages | "3 companies in [their vertical] did X" |
| **Authority** | Proposal | Certifications, expert advisors, "featured in" |
| **Liking** | All stages | Mirror language, reference their content, genuine curiosity |
| **Scarcity** | Close | "Q4 implementation slots: 2 left" (only if true) |
| **Loss Aversion** | Implication stage | "What's the cost of waiting 6 months?" |
| **Unity** | Relationship | "Built by [their role] for [their role]" |

## OBJECTION HANDLING: BRAIN-BASED

| Objection | System Triggered | Response Framework |
|-----------|------------------|-------------------|
| "Not interested" | System 1 (auto-reject) | "Totally fair. Most [roles] I talk to aren't either — until they see [specific proof]. Happy outcome]. Worth 2 min?" |
| "No budget" | System 2 (rationalization) | "Understood. The teams seeing ROI fastest actually reallocated from [waste area]. Can I show you the math?" |
| "Send me info" | System 1 (dismissal) | "Happy to. So I send the right thing — what's the one metric you'd need to see move?" |
| "We're good" | System 1 (status quo) | "Great to hear. Curious — how are you handling [specific edge case]?" |

## QUALITY CHECKLIST (Pre-Send)

- [ ] Subject line: 2-4 words, lowercase, no name
- [ ] Opener: Their world, specific observation
- [ ] Problem: Named in their language
- [ ] Proof: One number, one similar company
- [ ] CTA: Interest-based, one line reply
- [ ] Tone: Peer, not vendor (read aloud)
- [ ] Mobile preview: < 5 lines visible
- [ ] No: "hope you're well," "I'm [name] from," jargon, multiple links

## THE MINDSET SHIFT

> **"I'm not selling. I'm diagnosing whether this problem exists and whether I'm the right solver."**

This reduces neediness (System 1 threat signal) and increases authority (System 2 trust signal).
