- lista.com.br/busca?q=[TERMO]+[CIDADE]

- guiamais.com.br/busca?q=[TERMO]+[CIDADE]

## Dicas de Busca

1. Use aspas duplas (") para busca exata: "clinica odontologica"
2. Combine termos: "clinica odontologica" "Sao Paulo"
3. Use operadores: site:empresas.com.br "clinica odontologica" "Sao Paulo"
4. Varie os termos por cidade para encontrar mais resultados
5. Adicione delay de 2-5 segundos entre requisicoes
6. Nao acesse Google Maps ou Google Search diretamente
